Streaming Traffic Database Management

  • Role: Algorithm Engineer (Software Development)
  • Technologies Used: Kafka, Flink, HBase, Scala, Java
  • Description: Spearheaded the modernization of a large-scale streaming database system geared towards multimedia. Designed a unique logical framework that drastically reduced data stream sizes, leading to a 45% reduction in data payload. Collaborated with a team to introduce a new strategy that minimized latency by refining data flow and processing structures.

Fusion Perception for Autonomous Driving

  • Role: Developer and System Integrator
  • Technologies Used: ROS, Protobuf, C++
  • Description: Revitalized the Fusion Perception system for autonomous driving. Transitioned from ROS msg to Protobuf integrated with C++, enhancing data interchange, and boosting system responsiveness. Orchestrated the generation of binary release files to streamline deployment. Demonstrated adaptability and problem-solving skills by personally navigating challenging field deployments, where hands-on tuning and compiling of OpenCV parameters were done on-site, effectively addressing issues not solvable in an office setting.
